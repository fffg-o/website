Tailwind CSS v4 是一次重大的版本更新，带来了许多令人兴奋的新特性。

## 全新的引擎

Tailwind CSS v4 使用全新的引擎，构建速度提升了数倍。

## CSS-first 配置

新的版本采用 CSS-first 的配置方式，你可以直接在 CSS 中使用 `@theme` 指令来定义主题。

```css
@theme {
  --color-primary: #3b82f6;
}
```

## 性能提升

得益于新的引擎，构建速度相比 v3 有显著提升。
